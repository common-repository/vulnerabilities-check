== Description ==

Check the vulnerability of wordpress, plugins and themes.
The plugin is based on the service offered by http://wpvulndb.com. It makes a call to service for the current version of wordpress, for each theme and each plugin present for check for known vulnerabilities.
At the end, you can request a quote for assistance.

== Installation ==

1. Upload `vulnerabilities-check/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= How to use? =
Open the "Vulnerabilities Check" menu in wordpress admin and clic on "Check for vulnerabilities"
= What external service use? =
It use the service offered by http://wpvulndb.com
